# KlausurBotPro v0.1.1-exam

**Release-Datum:** 24.07.2026

## Zweck des Releases

Stabilisierter Klausurstand für Regelungstechnik 1.

## Enthaltene Hauptworkflows

- **Transferfunktion:** sichere rationale Eingabe, rohe und reduzierte Form,
  Pole, Nullstellen, Stabilität und qualitative Interpretation.
- **Frequenzbereich:** Frequenzgang, Bode, Standardglieder, Durchtritte,
  Reserven und Nyquist.
- **Stabilität:** Hurwitz- und Routh-Auswertung mit Parameterbedingungen im
  unterstützten Umfang.
- **Zeitbereich:** direkte und inverse Laplace-Transformation, DGL,
  Partialbruchzerlegung und ausgewählte Zeitantworten.
- **Zustandsraum:** charakteristisches Polynom, Eigenwerte,
  Zustandsstabilität und SISO-Übertragungsfunktion.
- **Reglerauslegung:** P-Auslegung über Phasenreserve, Ziegler–Nichols offen
  und geschlossen sowie Cohen–Coon für P, PI und PID im unterstützten Umfang.

## Wesentliche Phase-1-Verbesserungen

- aufgabenspezifische Zeitbereichsziele,
- eindeutige Zustandsstabilitätsausgabe,
- gemeinsame Bode-x-Achse,
- kompakte Frequenztabellen,
- sichere Rasterdiagnosen,
- explizite Hurwitz-Bedingungen und reduzierte Endmengen,
- kompaktes Routh-Schema,
- Transferfunktionsrundung und qualitative Polinterpretation,
- klausurtaugliche Reglerauslegung mit `K_T`,
- allgemeine Formel vor dem Einsetzen,
- exakte Werte vor Näherungen,
- verbesserte GUI-/Plaintext-/LaTeX-Synchronität.

## Installation und Start

```powershell
py -m venv .venv
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -e ".[dev]"
python -m klausurbotpro
```

Alternativ steht nach der Installation `klausurbotpro` als Konsolenskript zur
Verfügung.

## Bekannte Grenzen

- kein allgemeines CAS,
- kein beliebiger MIMO-Solver,
- keine allgemeine symbolische Parameteralgebra,
- keine Lead-/Lag-Auslegung,
- keine Totzeit-Frequenzrechnung,
- kein Standalone-Windows-Installer,
- fachliche Prüfung von Aufgabenstellung, Modellart, Voraussetzungen und
  Einheiten bleibt erforderlich.

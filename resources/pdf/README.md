# Lokale Original-PDFs

Der Nutzer legt offizielle Original-PDFs lokal in diesem Verzeichnis ab.

- Die Dateien werden nicht verändert.
- PDFs werden standardmäßig nicht in Git eingecheckt.
- Die Anwendung soll später ausschließlich relative Pfade verwenden.
- Dateinamen, Dokument-IDs und SHA-256-Prüfsummen werden später über ein
  Manifest verwaltet.

Diese README bleibt versioniert, auch wenn das Verzeichnis lokal keine PDFs
enthält.

# Lokale Referenz-Notebooks

Dieser Ordner ist für lokale fremde Referenzprogramme vorgesehen. Solche
Dateien dürfen ohne geklärte Lizenz nicht veröffentlicht oder committed
werden. Ihr Code wird nicht direkt übernommen; zulässig ist ausschließlich
die Nutzung für Ideen, unabhängig formulierte Testfälle und Funktionsvergleiche.
